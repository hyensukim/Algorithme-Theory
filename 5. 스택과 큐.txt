스택과 큐
- 스택 : 후입선출, 삽입과 삭제가 한쪽에서만 일어나는 특징. LIFO
	- top(스택 제일 위에 있는 위치)에서 삽입과 삭제가 일어남.
	- push : 데이터 top위치에 삽입
	- pop : top 위치 빼내서 보기
	- peek : top 위치 보기만
- 스택은 재귀함수 알고리즘 원리와 일맥상통함.
- 원리만 이해하자.

- 큐 : 선입선출 FIFO
	- rear : 큐 가장 끝
	- front : 큐에서 가장 앞
	- add : rear 부분에 새로운 데이터 삽입
	- poll : front 위치 데이터 빼내서 확인.
	- peek : front에 있는 데이터 확인 시 사용하는 연산.

*우선순위 큐
	- 들어간 순서와 상관없이 우선순위가 높은 데이터가 먼저 나오는 자료구조.(많이 나온다.)
	- 힙 자료구조(이진 트리 구조)
